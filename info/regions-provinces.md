# Spain Regions and provinces

Spain is divided into **17 Autonomous Communities**, which are further subdivided into **50 provinces**.
Additionally, there are **2 Autonomous Cities**.

Seven of the regions consist of only a single province (often called "uniprovincial" regions). Here is the complete list:

## Regions with Multiple Provinces

| Region (Autonomous Community) | Provinces |
| --- | --- |
| **Andalusia** | Almería, Cádiz, Córdoba, Granada, Huelva, Jaén, Málaga, Sevilla |
| **Aragon** | Huesca, Teruel, Zaragoza |
| **Basque Country** | Álava, Biscay (*Vizcaya*), Gipuzkoa (*Guipúzcoa*) |
| **Canary Islands** | Las Palmas, Santa Cruz de Tenerife |
| **Castile and León** | Ávila, Burgos, León, Palencia, Salamanca, Segovia, Soria, Valladolid, Zamora |
| **Castile-La Mancha** | Albacete, Ciudad Real, Cuenca, Guadalajara, Toledo |
| **Catalonia** | Barcelona, Girona, Lleida, Tarragona |
| **Extremadura** | Badajoz, Cáceres |
| **Galicia** | A Coruña, Lugo, Ourense, Pontevedra |
| **Valencian Community** | Alicante, Castellón, Valencia |

## Single-Province Regions

In these cases, the region and the province cover the same geographic area.

* **Asturias**
* **Balearic Islands**
* **Cantabria**
* **La Rioja**
* **Madrid**
* **Murcia**
* **Navarre**
